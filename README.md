# Kanban Guide

A visual guide to Kanban methodology, styled in the simpleshow aesthetic. This interactive HTML page demonstrates the principles of Kanban through a simulated project board.

## Features

- Interactive Kanban board with drag-and-drop functionality
- Work-in-progress (WIP) limits to prevent bottlenecks
- Visual alerts for lean principles
- Based on simpleshow's agile method explanation

## Preview

View the live demo: [Kanban Guide Demo](https://grinsteindavid.github.io/kankan/)

## How to Use

1. Open `index.html` in your web browser
2. Click "Break Down Project" to add tasks to the backlog
3. Drag tasks between columns: Subtasks → Active → Completed
4. Observe WIP limits and bottleneck warnings

## Technologies Used

- HTML5
- CSS3 (with Tailwind CSS)
- JavaScript (vanilla)
